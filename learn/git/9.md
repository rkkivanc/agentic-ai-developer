## What Have I Learned?

I learned that `git push` uploads local commits to a remote. `-u` stands for upstream, you should use once per branch on first push so you connect your branch to github's branch that you selected. After that you can only use `git push`.

First: `git push -u origin main` (if you want to change origin's branch name you can do something like `git push -u origin main:something-else`)

Then: `git push`
