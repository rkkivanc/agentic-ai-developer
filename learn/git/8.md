## What Have I Learned?

I learned how `git remote` ACTUALLY works! I had been thinking that git remote is only a command that links my local repo with github. Now, I know that it works like a contact app and origin is the usual nickname for my main remote URL. It is something like a contact on our phone. We can add more than one. As an example, we can refer something with origin while we refer another thing with a nickname backup.

`git remote add` only saves the link, it does not upload any code yet or something different.

`git remote -v` shows your saved contacts and their URLs. `-v` stands for `verbose`. If you do not write `-v` then you will see only the name of remote (origin). when you write `-v` you will see something like:
  
```
origin	https://github.com/rkkivanc/git-day1-practice (fetch)
origin	https://github.com/rkkivanc/git-day1-practice (push)
```
