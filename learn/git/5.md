## What Have I Learned?

I learned that a branch is an independent line of development (like a new workspace but not actually), so I can try new work without harming `main`.

I learned `git branch` creates a branch but does not switch to it. Newer: `git switch` or classic: `git checkout` moves me onto that branch.

I learned commits go on the branch I am currently on, and `git log --graph --oneline --all` shows where each branch points. There are shapes such as |, /, \, * on graph. I only saw * because the branch was only one commit away from the main and there was no new commits on main.
