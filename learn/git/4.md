## What Have I Learned?

I learned that git ignores what is written in `.gitignore`, and it is built-in convention of git. The file name must be exactly same.

I learned how `git check-ignore` works and what `-v` means (verbose, it also shows the matching pattern from .gitignore).

In addition, I learned shell tools such as `echo`, `>`, `touch`, `cat`, `>>`, `EOF`.
