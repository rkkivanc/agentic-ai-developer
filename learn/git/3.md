## What Have I Learned?

I learned how to use `git status` and `git log`. I explored different `git log` options like `--oneline` and `--graph`, and found out that the order of these flags does not matter. I also learned what `(HEAD -> main)` means in the commit log—it shows up on the most recent commit and indicates that `HEAD` (your current position in the repository) is pointing to the `main` branch, which is the branch you are currently on.
