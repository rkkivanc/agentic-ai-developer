## 📚 Dictionary for Developers

*   **Apollo Client:** A comprehensive GraphQL client with cache, local state, and React integrations.
*   **urql:** A lightweight GraphQL client emphasizing simplicity and extensible exchanges.
*   **Cache Policy:** Rules determining when cached query results are reused or refetched.
*   **Optimistic Update:** Updating the UI before the server confirms a mutation succeeded.
*   **Normalized Cache:** Storage that deduplicates objects by ID across queries.
