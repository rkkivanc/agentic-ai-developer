## 📚 Dictionary for Developers

*   **Cursor Pagination:** Paging with opaque cursors instead of numeric offsets.
*   **Relation Field:** A field returning related objects, e.g., `author.posts`.
*   **Max Page Size:** An upper bound on items returned per request.
*   **PageInfo:** Metadata describing whether more pages exist in Relay connections.
*   **Bounded Query:** A list fetch that cannot return unlimited rows.
