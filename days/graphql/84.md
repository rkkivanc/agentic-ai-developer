## 📚 Dictionary for Developers

*   **Cache Invalidation:** Removing or updating cached data when underlying values change.
*   **TTL (Time To Live):** Duration after which cached data expires and must be refreshed.
*   **CDN (Content Delivery Network):** Edge network that can cache HTTP responses closer to users.
*   **Query Plan:** Analysis of which resolvers run and how long they take for an operation.
*   **Performance Budget:** Agreed limits on latency, payload size, or resolver cost for operations.
