## 📚 Dictionary for Developers

*   **Subscription:** A long-lived GraphQL operation streaming events from server to client.
*   **WebSocket:** A persistent bidirectional protocol commonly used for GraphQL subscriptions.
*   **Pub/Sub:** Publish-subscribe messaging where publishers emit events subscribers consume.
*   **SSE (Server-Sent Events):** A one-way HTTP streaming alternative for realtime updates.
*   **Event Payload:** The data sent to subscribers when an event occurs.
