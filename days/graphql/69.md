## 📚 Dictionary for Developers

*   **Unit Test:** A test of an isolated function such as a resolver with mocked dependencies.
*   **Integration Test:** A test that runs operations against a real or in-memory GraphQL server.
*   **Mock:** A test double replacing databases or services with predictable responses.
*   **Snapshot Test:** A test comparing current schema or response output to a saved baseline.
*   **Test Fixture:** Reusable setup data and helpers for consistent test environments.
