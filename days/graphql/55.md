## 📚 Dictionary for Developers

*   **Subscription Field:** A schema field clients subscribe to for event streams.
*   **Publish:** Server emitting an event to active subscribers.
*   **WebSocket Client:** A tool or library maintaining the subscription connection.
*   **Polling Fallback:** Periodic queries used when push transport is unavailable.
*   **Realtime UX:** UI updating automatically when subscription events arrive.
