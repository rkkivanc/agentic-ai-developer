## 📚 Dictionary for Developers

*   **Query Depth Limit:** A cap on how deeply nested a query's selection set may go.
*   **Complexity Score:** A numeric cost assigned to fields to limit expensive queries.
*   **Rate Limiting:** Restricting how many requests a client may send in a time window.
*   **Persisted Query:** A server-known query ID sent instead of the full query string.
*   **Allow List:** A security approach where only pre-approved operations are executed.
