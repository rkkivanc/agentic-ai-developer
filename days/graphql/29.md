## 📚 Dictionary for Developers

*   **Apollo Server:** A popular Node.js GraphQL server with plugins, subscriptions, and federation support.
*   **GraphQL Yoga:** A lightweight, standards-focused GraphQL server for Node and other runtimes.
*   **MVP (Minimum Viable Product):** The simplest version of an API that demonstrates core value.
*   **Data Source:** Any database, REST service, or in-memory store backing your resolvers.
*   **CRUD:** Create, Read, Update, Delete—the basic data operations most APIs support.
