## 📚 Dictionary for Developers

*   **Pagination:** Splitting large lists into pages using cursors or offsets.
*   **Connection:** A Relay-style pattern with `edges`, `nodes`, and `pageInfo` for cursor pagination.
*   **Cursor:** An opaque token marking a position in a paginated list.
*   **Relation:** A link between types, such as one author having many posts.
*   **PageInfo:** Metadata describing whether more pages exist before or after the current window.
