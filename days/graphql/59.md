## 📚 Dictionary for Developers

*   **N+1 Problem:** Performance issue where one query triggers N additional queries for related items.
*   **DataLoader:** A utility that batches and caches loads within a single request.
*   **Batch Function:** A function DataLoader calls with a list of keys to fetch in one round trip.
*   **Per-Request Cache:** A cache scoped to one HTTP request, preventing cross-user data leaks.
*   **Eager Loading:** Fetching related data proactively to avoid repeated round trips.
