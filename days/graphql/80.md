# Day 80: Tooling, Codegen & DX - Practice Project & Review

## 📝 Today's Tasks

1.  **Codegen Setup:** Generate TypeScript types and hooks from your schema and operations.
2.  **ESLint GraphQL:** Add lint rules for queries and SDL in CI.
3.  **npm Scripts:** Add `codegen`, `lint`, and `test` scripts to package.json.
4.  **DX Review:** Time how long a new developer takes to run queries with your docs and scripts.

## 📖 Task Explanations

Today improves daily workflow on your project. Codegen eliminates hand-written operation types; lint catches invalid fields before runtime.

Measure DX informally—can you clone, install, codegen, and query in under ten minutes?

## 💡 What You'll Gain

By the end of today, you will have codegen, lint, and scripts integrated into your GraphQL project workflow.

## 📚 Dictionary for Developers

*   **GraphQL Code Generator:** Tool producing typed client code from schema and documents.
*   **GraphQL ESLint:** Lint rules for query documents and schema files.
*   **npm Script:** Package.json command wrapping codegen, test, or lint tasks.
*   **Typed Hook:** React hook generated with precise operation result types.
*   **Developer Onboarding:** Experience of a new engineer running the project locally.
