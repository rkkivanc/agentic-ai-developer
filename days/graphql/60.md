## 📚 Dictionary for Developers

*   **N+1 Fix:** Replacing per-item queries with one batched fetch.
*   **Batch Loader:** DataLoader function resolving many IDs in one call.
*   **Query Count:** Number of database round trips per GraphQL request.
*   **Singleton Anti-pattern:** Sharing a DataLoader across requests—unsafe.
*   **Performance Win:** Measurable latency or query reduction after optimization.
