## 📚 Dictionary for Developers

*   **Code Generation:** Automatically producing TypeScript types and hooks from a GraphQL schema.
*   **GraphQL Code Generator:** A popular tool generating typed client and server code from SDL.
*   **GraphQL ESLint:** Lint rules enforcing query and schema conventions in CI and editors.
*   **Typed Document Node:** A query representation carrying TypeScript types for compile-time safety.
*   **Developer Portal:** Documentation site where consumers explore and test your GraphQL API.
