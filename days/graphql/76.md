# Day 76: Tooling, Codegen & DX - GraphQL Code Generator Setup

## 📝 Today's Tasks

1.  **Install Codegen:** Add `@graphql-codegen/cli` and TypeScript plugins to your project.
2.  **Point at Schema:** Configure schema from SDL files or introspection URL.
3.  **Add Operation Documents:** Place `.graphql` queries and mutations for typed hooks.
4.  **Run `graphql-codegen`:** Generate types and verify they match your operations.

## 📖 Task Explanations

GraphQL Code Generator eliminates hand-written TypeScript types for operations. It reads your schema and documents, producing typed hooks for Apollo Client or urql.

Commit generated output or regenerate in CI—pick one workflow and document it for the team.

## 💡 What You'll Gain

By the end of today, you will configure GraphQL Code Generator producing typed operation results for your client.

## 📚 Dictionary for Developers

*   **GraphQL Code Generator:** Toolchain generating typed client and server artifacts from schema.
*   **Typed Operations:** TypeScript types mirroring variables and result shapes.
*   **Codegen Config:** YAML file listing schema sources, documents, and plugins.
*   **Client Preset:** Plugin bundle generating hooks for Apollo or urql.
*   **Generated Types:** Output TypeScript declarations kept in sync with schema.
