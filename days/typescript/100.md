## 📚 Dictionary for Developers

*   **Matrix Build:** CI running jobs across multiple Node versions or configurations simultaneously.
*   **Quality Gate:** Automated checks blocking merge when tests, lint, or types fail.
*   **Capstone Delivery:** The final polished project demonstrating curriculum mastery.
*   **Portfolio Reflection:** A written summary of skills gained and work produced.
*   **Professional TypeScript:** Engineering with types, validation, testing, CI, and clear architecture.
