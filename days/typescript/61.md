## 📖 Task Explanations

Conditional types select types based on relationships—`T extends Array<infer U> ? U : T` extracts element types. They power many built-in utilities and advanced library typings. Distribution over unions means `T extends U ? X : Y` applies per union member when `T` is a union.

These types do not exist at runtime—they are compile-time logic. Reading them takes practice; writing them takes more. Start by decoding existing utilities before inventing your own.

Day 61 opens Advanced Types II—the type-level programming that distinguishes library authors from application developers.

## 📚 Dictionary for Developers

*   **Conditional Type:** `T extends U ? X : Y`—selecting types based on a condition.
*   **Distributive Conditional:** A conditional distributing over union members when `T` is a union.
*   **Type-Level Logic:** Compile-time branching producing different types per input.
*   **`Exclude<T, U>`:** Removes types assignable to `U` from union `T`.
*   **Type Computation:** Deriving new types from existing ones without runtime code.
