# Day 11: Inheritance & Polymorphism - Inheritance Basics

## 📝 Today's Tasks

1.  **Create a Base Class:** Define a parent with shared fields/methods.
2.  **Extend It:** Create 2 subclasses that inherit and add specialized behavior.
3.  **Reuse Check:** Call inherited methods from subclass instances.
4.  **Hierarchy Caution:** Keep the hierarchy shallow (one level is enough today).

## 📖 Task Explanations

Inheritance shares implementation across related types. It is powerful and easy to overuse. Today you practice a shallow hierarchy: reuse without building a fragile tower of classes.

## 💡 What You'll Gain

You will build a simple inheritance hierarchy and reuse base behavior in subclasses.

## 📚 Dictionary for Developers

*   **Inheritance:** Creating a new class based on an existing class.
*   **Base / Superclass:** The parent class providing shared members.
*   **Subclass / Derived Class:** A class that extends a base class.
*   **Fragile Base Class:** A problem where changes to a parent unexpectedly break children.
