## 📚 Dictionary for Developers

*   **Flaky Test:** A test that intermittently fails without consistent code changes.
*   **Quarantine:** Temporarily isolating flaky tests while investigating root causes.
*   **Test Documentation:** Instructions explaining which tests exist and how to run them.
*   **Regression Suite:** Tests run on every change to detect broken behavior early.
*   **Trustworthy CI:** A pipeline whose failures indicate real problems, not noise.
